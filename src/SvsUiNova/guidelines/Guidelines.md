## Design System Package

This package provides:
- Svenska Spel brand fonts (Arena Pro, Plan Pro)
- Component library with Figma Code Connect mappings
- Design tokens via CSS custom properties
- Typography classes

## Installation

This design system is designed to work seamlessly with Figma Make. When you import components from Figma that use SvsUiNova, the setup will be handled automatically if the Make Kit infrastructure has been properly configured with the font loading fix.
