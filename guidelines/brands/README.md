# SvS UI (Nova) - Brands

This directory contains detailed documentation for brand identity elements in the Svenska Spel Design System.

Brand documentation covers visual identity, brand assets, and guidelines that ensure consistent brand expression across all digital products.

---

## Available Brand Topics

- [Color](./Color.md) - Brand color palette, usage guidelines, and accessibility standards
- [Logotype](./Logotype.md) - Logotype system, Vinnarkuben symbol, wordmark, and usage guidelines
- [Typography](./Typography.md) - Custom typeface family, Svenska Spel Arena and Plan usage guidelines
- [Imagery](./Imagery.md) - Photography style, composition principles, and visual storytelling guidelines
- [Design Elements](./DesignElements.md) - Word boxes, price labels, tags, and graphic design elements
- [Icons and Illustrations](./IconsAndIllustrations.md) - Simple icons, pictograms, and illustrations system

---

## Adding New Brand Topics

When importing a new brand topic from Figma:

1. Create a new `.md` file named after the topic (e.g., `Logo.md`, `Typography.md`)
2. Follow the established documentation structure
3. Add a link to the topic in this README
4. Add a link to the topic in the main `Guidelines.md` file
