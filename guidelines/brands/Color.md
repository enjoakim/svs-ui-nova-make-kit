# Color

The Svenska Spel brand color palette and usage guidelines. Red is our hero color, complemented by warm neutrals and business area accent colors.

---

## Philosophy

When you think Svenska Spel, you should think red. Our main color is red with complementary red tones. This allows us to be red in a dynamic way. Red as in high energy and pulse, but also red as in warm and caring.

**Core Principles:**
- **The hero is always our primary red** - The main red (#ED0000) is the foundation
- **Red for CTA or activity** - Use red to draw attention and drive action
- **White, beige, and black as base** - Creates a distinct expression with dark mode capability

## Color Hierarchy

### Primary Hierarchy
1. **Svenska Spel Red (#ED0000)** - Hero color, primary CTAs, brand moments
2. **White / Beige Light (#FFFFFF / #FAF6F3)** - Backgrounds, surfaces
3. **Black (#1B1918)** - Text, dark mode

### Secondary Hierarchy
- **Dark Reds (#A20020, #62001D)** - Depth, accents, variations
- **Beige Medium (#E5DFDA)** - Subtle backgrounds, dividers
- **Business Colors (Blue/Green)** - Category differentiation only

## Color Usage Guidelines

### Do's

**Primary Red:**
- ✓ Use for primary CTAs and important actions
- ✓ Use for brand moments and key messaging
- ✓ Use to create energy and draw attention
- ✓ Pair with white or beige for maximum impact

**Neutrals:**
- ✓ Use beige tones for warm, approachable backgrounds
- ✓ Use white for clean, fresh surfaces
- ✓ Use black for strong text contrast
- ✓ Layer neutrals to create depth

**Business Colors:**
- ✓ Use to clearly differentiate Sport & Casino from Turspel
- ✓ Use consistently within each business area
- ✓ Use for category navigation and identification

---

### Don'ts

**Primary Red:**
- ✗ Don't use as large background areas (except key moments)
- ✗ Don't combine with business area colors for CTAs
- ✗ Don't dilute with too many tints

**Neutrals:**
- ✗ Don't use too many neutral tones in one composition
- ✗ Don't create muddy contrasts
- ✗ Don't lose warmth by overusing pure black/white

**Business Colors:**
- ✗ Don't use blue and green together
- ✗ Don't use business colors for primary CTAs
- ✗ Don't mix business colors across categories

## Accessibility

### Contrast Ratios

**Text on Backgrounds:**
- Black (#1B1918) on White (#FFFFFF): 19.37:1 ✓ AAA
- Black (#1B1918) on Beige Light (#FAF6F3): 18.23:1 ✓ AAA
- White (#FFFFFF) on Primary Red (#ED0000): 3.99:1 ✓ AA Large
- White (#FFFFFF) on Dark Red 1 (#A20020): 7.36:1 ✓ AAA
- White (#FFFFFF) on Dark Red 2 (#62001D): 12.85:1 ✓ AAA

**Recommendations:**
- Use Primary Red (#ED0000) with white text for large text only (18pt+)
- Use Dark Red tones for better text contrast when needed
- Always verify contrast meets WCAG 2.1 AA standards minimum
- Aim for AAA standards where possible

## Dark Mode

### Color Adaptations

**Background:**
- Light mode: White (#FFFFFF) / Beige Light (#FAF6F3)
- Dark mode: Black (#1B1918)

**Surface:**
- Light mode: Beige Light (#FAF6F3) / Beige Medium (#E5DFDA)
- Dark mode: Lighter variations of black, subtle grays

**Primary Red:**
- Remains consistent across light and dark modes
- May be slightly desaturated in dark mode for comfort

**Business Colors:**
- Remain consistent for category identification
- Adjust brightness for dark backgrounds

## Color Combinations

## Implementation Notes

### Design Tokens

All colors should be implemented as design tokens in the design system:

```
--color/brand/primary: #ED0000
--color/brand/dark-1: #A20020
--color/brand/dark-2: #62001D

--color/neutral/beige-light: #FAF6F3
--color/neutral/beige-medium: #E5DFDA
--color/neutral/black: #1B1918
--color/neutral/white: #FFFFFF

--color/business/sport-casino: #0071DB
--color/business/turspel: #00823D
```

See `/src/SvsUiNova/styles.css` for the complete token implementation.
