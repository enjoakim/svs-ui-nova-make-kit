# Typography

The Svenska Spel custom typeface family featuring Svenska Spel Arena for display and Svenska Spel Plan for body text.

---

## Philosophy

Our typeface is specially designed just for us. Proud and kind. Energetic with a lot of feeling. A voice with the confidence that suits a market leader. Powerful to highlight impact and the greater feeling.

**Core Principles:**
- **Developed for both communication and service** - Works across all contexts
- **Can speak about everything from responsible gaming to jackpots** - Versatile and adaptable
- **Custom-designed exclusively for Svenska Spel** - Unique brand expression

---

## Typeface Family

Our typeface family consists of the uppercase display typeface **Svenska Spel Arena** and the body typeface **Svenska Spel Plan**.

### Svenska Spel Arena Pro

**Purpose:** Display typeface for headlines and impactful communication

**Characteristics:**
- Uppercase only (all caps)
- Bold weight
- Designed for short, powerful headlines
- Our main voice in communication

**Available Weights:**
- **Bold** - The only weight available

**Usage:**
- Main headlines
- Hero text
- Campaign headers
- Attention-grabbing moments
- Short, impactful messages

**Best Practices:**
- Use for shorter headlines in uppercase
- Creates energy and impact
- Pairs perfectly with Svenska Spel Plan for body text

---

### Svenska Spel Plan Pro

**Purpose:** Body typeface for text content and readability

**Characteristics:**
- Mixed case (uppercase and lowercase)
- Multiple weights for hierarchy
- Designed for extended reading
- Creates structure and clarity

**Available Weights:**
- **Regular** - Body text and captions
- **Medium** - Subheadings and introductory text
- **Bold** - Emphasis and strong hierarchy

**Italic Variants:**
- All weights have corresponding italic styles

**Usage by Weight:**

**Regular:**
- Body text / paragraph text
- Captions
- General content
- Extended reading

**Medium:**
- Subheadings
- Introductory text / lead paragraphs
- Section headers
- Emphasized content

**Bold:**
- Strong emphasis
- Important callouts
- Key information highlights

## File Formats

Our typeface is available in three different file formats depending on user and use case.

### 1. OTF (OpenType Font) - Pro Version

**File Extension:** `.otf`

**Naming:** Svenska Spel Arena Pro, Svenska Spel Plan Pro

**Intended For:**
- Layout and design programs
- Adobe Creative Suite (Photoshop, Illustrator, InDesign)
- Figma
- Sketch
- Professional design tools

**Operating Systems:**
- Windows
- macOS

**When to Use:**
This is the professional version with the most features and best quality for design work.

---

### 2. TTF (TrueType Font) - Basic Version

**File Extension:** `.ttf`

**Naming:** Svenska Spel Arena, Svenska Spel Plan (without "Pro")

**Intended For:**
- Microsoft Office applications
- Microsoft PowerPoint (primary use)
- Microsoft Word
- Office productivity tools

**Operating Systems:**
- Windows
- macOS

**When to Use:**
If you're working in Microsoft PowerPoint or Microsoft Word, you should use the basic version instead of the Pro version.

---

### 3. WOFF / WOFF2 (Web Open Font Format)

**File Extension:** `.woff`, `.woff2`

**Intended For:**
- Websites
- Web applications
- Digital environments only

**When to Use:**
Web font versions are specifically created for use on websites and should only be integrated in digital environments.

**Implementation:**
These fonts are optimized for web performance and cross-browser compatibility.

---

## Fallback Fonts

When our custom typefaces are not available, such as in simpler digital tools or programs, use these fallback fonts.

### For Svenska Spel Arena

**Fallback:** Tahoma Bold

**When to Use:**
- When Svenska Spel Arena is not available
- In simpler digital tools
- In programs like PowerPoint (if custom fonts can't be installed)
- For headlines and display text

---

### For Svenska Spel Plan

**Fallback for Body Text:** Arial Regular

**Fallback for Subheadings:** Arial Bold

**When to Use:**
- When Svenska Spel Plan is not available
- In simpler digital tools
- In programs like PowerPoint (if custom fonts can't be installed)

**Application:**
- Arial Regular for body text
- Arial Bold for introductory paragraphs and subheadings

## Usage Guidelines

### Design with Our Typeface

We can use our typeface in different ways – from a clean, stripped-down design where the content takes center stage, to expressive layouts with large headlines, images, and design elements. This way we can adapt the expression and create the right feeling depending on context.

### Typography Hierarchy

**Level 1 - Main Headline:**
- **Typeface:** Svenska Spel Arena Pro Bold
- **Style:** UPPERCASE
- **Use:** Hero moments, main page headers

**Level 2 - Section Headline:**
- **Typeface:** Svenska Spel Plan Pro Medium or Bold
- **Style:** Sentence case or Title Case
- **Use:** Section headers, card titles

**Level 3 - Subheading:**
- **Typeface:** Svenska Spel Plan Pro Medium
- **Style:** Sentence case
- **Use:** Subsections, introductory text

**Body Text:**
- **Typeface:** Svenska Spel Plan Pro Regular
- **Style:** Sentence case
- **Use:** Paragraphs, descriptions, content

**Captions:**
- **Typeface:** Svenska Spel Plan Pro Regular
- **Style:** Sentence case
- **Use:** Image captions, metadata, small text

## Don'ts

Use only the typefaces in the variants and combinations specified above.

### Typography Don'ts

**❌ Don't:**
- Use Svenska Spel Arena in lowercase
- Use Svenska Spel Arena for long paragraphs or body text
- Mix Svenska Spel fonts with other brand typefaces (except fallbacks)
- Create custom weights by artificially bolding or thinning
- Stretch or distort the typefaces
- Use Svenska Spel fonts in unapproved combinations
- Apply effects like outlines, shadows, or distortions (unless part of approved design patterns)

**✓ Do:**
- Use Svenska Spel Arena in uppercase for headlines
- Use Svenska Spel Plan for all body text
- Maintain proper hierarchy with weight variations
- Use italic styles when available for emphasis
- Follow the specified file format guidelines

## Character Set

Both Svenska Spel Arena and Svenska Spel Plan include:

- **Latin alphabet** - A-Z, a-z (Plan only for lowercase)
- **Numbers** - 0-9
- **Swedish characters** - Å, Ä, Ö, å, ä, ö
- **Common punctuation** - . , ! ? : ; " ' - ( ) / etc.
- **Special characters** - @, &, %, €, etc.

The fonts support Swedish and other European languages.

## Accessibility Considerations

### Readability

**Minimum Sizes:**
- **Body Text:** Minimum 14px (desktop), 16px (mobile) recommended
- **Captions:** Minimum 12px
- **Headlines:** Scale appropriately for hierarchy

### Contrast

Always ensure sufficient contrast between text and background:
- **Body text on light:** Use black (#1B1918) or dark gray
- **Body text on dark:** Use white (#FFFFFF) or light beige (#FAF6F3)
- Follow WCAG 2.1 AA standards minimum (4.5:1 for body text)

### Line Height

**Body Text:**
- Recommended line height: 1.4-1.6
- Comfortable reading for extended text

**Headlines:**
- Tighter line height: 0.9-1.1
- Creates impact and visual hierarchy

## Technical Implementation

### CSS Font Stack

```css
/* Headlines - Display */
font-family: 'Svenska Spel Arena Pro', 'Tahoma', sans-serif;
font-weight: bold;
text-transform: uppercase;

/* Body Text */
font-family: 'Svenska Spel Plan Pro', 'Arial', sans-serif;
font-weight: 400; /* Regular */

/* Subheadings */
font-family: 'Svenska Spel Plan Pro', 'Arial', sans-serif;
font-weight: 500; /* Medium */

/* Strong Emphasis */
font-family: 'Svenska Spel Plan Pro', 'Arial', sans-serif;
font-weight: 700; /* Bold */
```

### Design Tokens

Typography tokens are defined in `/src/SvsUiNova/styles.css`:

```
--font-family/display: 'Svenska Spel Arena Pro'
--font-family/body: 'Svenska Spel Plan Pro'

--font-size/hero: [size]
--font-size/h1: [size]
--font-size/h2: [size]
--font-size/body: 16px
--font-size/caption: 14px
```

## Exceptions

In exceptional cases, when our regular typefaces are not available, fallback fonts can be used. Fallback fonts are only used when no other alternatives are possible.

**Never use fallback fonts in:**
- Leading external communication
- Brand campaigns
- Major marketing materials
- Official brand touchpoints

**Acceptable for fallback fonts:**
- Internal documents
- Quick presentations
- Emergency situations
- Systems where custom fonts cannot be installed
