# SvS UI (Nova) - Writing

This directory contains detailed documentation for writing guidelines, content patterns, and voice & tone principles in the Svenska Spel Design System.

Writing documentation covers language usage, messaging patterns, and content guidelines that ensure consistent, clear, and user-friendly communication across all digital products.

---

## Available Writing Topics

- [Tone and Voice](./ToneAndVoice.md) - Svenska Spel's tone of voice, personality, and writing style guidelines (Swedish)
- [Tone Scale](./ToneScale.md) - How tone adapts across different contexts while maintaining core personality (Swedish)
- [Writing Rules](./WritingRules.md) - Comprehensive writing conventions, formatting standards, and style guidelines (Swedish)
- [Vocabulary](./Vocabulary.md) - Alphabetical glossary of Svenska Spel-specific terms with correct spelling and usage (Swedish)

---

## Adding New Writing Topics

When importing a new writing topic from Figma:

1. Create a new `.md` file named after the topic (e.g., `VoiceAndTone.md`, `ContentGuidelines.md`)
2. Follow the established documentation structure
3. Add a link to the topic in this README
4. Add a link to the topic in the main `Guidelines.md` file
